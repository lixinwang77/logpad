# Logpad 版本历史

## 版本规范

采用语义化版本 `MAJOR.MINOR.PATCH`（例如 `1.2.3`）：

| 变更类型 | 版本号变化 | 示例 |
|----------|------------|------|
| Bug 修复 | PATCH +1 | 1.0.0 → 1.0.1 |
| 新功能（向后兼容） | MINOR +1，PATCH 归零 | 1.0.1 → 1.1.0 |
| 不兼容的重大变更 | MAJOR +1，MINOR/PATCH 归零 | 1.1.0 → 2.0.0 |

每次发版还需递增 **Build 号**（`CURRENT_PROJECT_VERSION`），与版本号同步更新。

### 发版步骤

1. 修改 `Logpad/Logpad.xcodeproj/project.pbxproj` 中的 `MARKETING_VERSION` 和 `CURRENT_PROJECT_VERSION`（Debug / Release 两处都要改）
2. 在本文件「变更记录」中追加条目
3. 提交代码

## 变更记录

### 1.3.1 (2026-05-29)

- 修复在日志中选中文字后点击其他行时，旧选区残留为灰色框的问题（焦点切换到新行时清除上一行选区）

### 1.3.0 (2026-05-29)

- 当前导航到的搜索匹配以更深的黄橙色高亮，区别于其余普通黄色匹配，便于辨认聚焦位置；点击过滤结果列表中的匹配也会聚焦并加深

### 1.2.1 (2026-05-29)

- 修复同一行存在多个匹配时只高亮第一个、且 `Enter`/箭头无法跳转到同行后续匹配的问题（现在每个匹配都高亮并可逐个导航）

### 1.2.0 (2026-05-29)

- 搜索框右侧新增上/下箭头按钮，点击跳转上一条/下一条匹配，鼠标悬停显示对应快捷键（`⇧↵` / `↵`），无结果时禁用

### 1.1.2 (2026-05-29)

- 修复空状态/错误页本地化 key 不匹配（`dragDropHint` / `openFile` / `tryAgain` 显示为原始 key），并将拖放提示改为含 `⌘O` 快捷键的完整文案

### 1.1.1 (2026-05-29)

- 搜索框占位文本显示快捷键提示（`Search (⌘F)` / `搜索 (⌘F)`）

### 1.1.0 (2026-05-29)

- 新增外部文件修改检测：文件被外部写入/删除/替换时显示提示条，可一键重新加载（`Cmd+R`，保持滚动位置）或忽略

### 1.0.0 (2026-05-29)

- 初始发布：文件浏览、搜索高亮、文本标记、分屏布局、跳转行号、中英文界面
- 添加应用版本号与「关于 Logpad」窗口
